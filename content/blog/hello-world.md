## Welcome!

Hey there! 👋 I'm Tanmay, and this is the very first post on my blog. I've been meaning to start writing for a while now, and here we are.

## What to Expect

I plan to write about things I work on day-to-day:

- **Infrastructure as Code** — Terraform, Pulumi, and the art of managing cloud resources declaratively.
- **Kubernetes** — Lessons from running production clusters, debugging tricky issues, and building platforms on top of K8s.
- **CI/CD Pipelines** — Designing efficient pipelines that teams actually enjoy using.
- **Backend Engineering** — Building APIs, working with databases, and writing code that scales.

## Why Blog?

I believe that writing forces you to think clearly. If I can explain something in a blog post, I probably understand it well enough. And if someone else finds it useful — even better.

Stay tuned for more posts. 🚀
